
# Smart Home Control Panel

## Overview
A responsive web app built with React JS, simulating a mobile smart home interface. Features real-time weather for Delhi via WeatherAPI.com, room/device management, and bonuses like dark mode, persistence, and animations.

## Setup
1. `npm install`
2. Add `VITE_WEATHER_API_KEY` to `.env`
3. `npm run dev`

## Features
- Dynamic greeting & auto dark mode
- Weather for Delhi with °C/°F toggle
- Horizontal room scroll with smooth transitions
- Device toggles with persistence (localStorage) & icon theming
- Mocked energy summary with tips
